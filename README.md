# Calendario App de Eventos

Visita el Proyecto: https://calendario-app-react.herokuapp.com

## Tecnologias de Back-End:

- Node.js
- Express
- Mongo DB

## Tecnologias de Front-End:

- React v17.x
- Redux
- React-Router-DOM
- react-big-calendar
- react-datetime-picker
- react-bootstrap
- react-modal
- redux-thunk
- sweetalert
- node-sass
